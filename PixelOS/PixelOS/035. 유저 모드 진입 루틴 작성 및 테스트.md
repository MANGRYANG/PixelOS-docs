# 변경 사항

## 코드

**user_mode.asm**

```ASM
[BITS 32]

section .text

global jump_usermode

; 현재 사용자 코드/데이터 세그먼트 셀렉터 (RPL = 3) 
%define USER_CODE_SEG 0x1B
%define USER_DATA_SEG 0x23

;------------------------------------------------------------------
; void jump_usermode(uint32_t user_eip, uint32_t user_esp);
;------------------------------------------------------------------
jump_usermode:
    cli
	
    ; 유저 데이터 세그먼트를 미리 로드 (CPL0에서 RPL=3 셀렉터 로드가 가능함)
    mov ax, USER_DATA_SEG
    mov ds, ax
    mov es, ax
    mov fs, ax
    mov gs, ax
	
    ; 함수 인자 미리 확보
    mov ecx, [esp + 4] ; user_eip
    mov edx, [esp + 8] ; user_esp
	
    ; iretd를 위한 스택 프레임 구성
    ; SS, ESP, EFLAGS, CS, EIP 순서로 푸시
    
    push dword USER_DATA_SEG        ; SS
	
    push edx                        ; ESP (user_esp)
	
    pushfd
    pop eax
    or eax, 0x200                   ; IF bit 설정
    push eax                        ; EFLAGS
    
    push dword USER_CODE_SEG        ; CS
    
    push ecx                        ; EIP (user_eip)
    
    iretd

section .usertext

global usermode_entry
extern user_test_main

;------------------------------------------------------------------
; void usermode_entry(void);
;------------------------------------------------------------------
usermode_entry:
    mov ax, USER_DATA_SEG
    mov ds, ax
    mov es, ax
    mov fs, ax
    mov gs, ax
	
    call user_test_main             ; 실제 유저 모드 테스트 함수 호출

.hang:
    jmp .hang
```

**커널 모드(`CPL 0`)에서 유저 모드(`CPL 3`)로 전환하는 진입 루틴** 및 **유저 모드로 전환한 직후 실행할 엔트리**가 구현된 어셈블리 코드입니다.

`jump_usermode`는 커널에서 호출되는 루틴으로, 다음과 같이 동작합니다.

1. `cli` 명령을 통해 인터럽트를 잠시 비활성화합니다.
2. 데이터 세그먼트 레지스터들에 **유저 데이터 세그먼트 셀렉터**를 로드합니다.
3. 함수 인자를 미리 확보합니다.
4. `iretd`가 사용할 복귀 스택 프레임(`SS`, `ESP`, `EFLAGS`, `CS`, `EIP`)을 구성합니다.
5. `iretd`를 호출합니다.

`x86 CPU`는 `iretd`을 실행할 때 스택에 들어 있는 `CS, SS`의 권한 레벨 정보를 통해 CPL을 설정하므로, 해당 루틴을 통해 `CPL 3`으로 진입할 수 있습니다.

`usermode_entry`는 **유저 모드로 전환한 이후 실행할 엔트리 루틴**으로, `.usertext` 섹션에 위치합니다.

유저 데이터 세그먼트를 재설정하고 **유저 테스트 함수(`user_test_main`)를 실행**하도록 작성하였습니다.


**kernel.c**

```C
...

// 유저 모드 진입 함수
extern void jump_usermode(uint32_t user_eip, uint32_t user_esp);
extern void usermode_entry(void);

...

void kernel_main(void)
{
	...
	window_put_string(
	    g_testwin,
	    "[RING3] jump to user mode...\n",
	    COLOR_BROWN
	);
    compositor_compose();
	
    jump_usermode(
	    (uint32_t)usermode_entry,
	    (uint32_t)(g_user_test_stack + 4096)
    );
	
    for (;;)
    {
        __asm__ volatile ("hlt");
    }
	...
}

```

`user_mode.asm`에서 구현한 루틴을 커널 코드에서 사용할 수 있도록 선언하였습니다.

커널 메인 함수에서 `jump_usermode`를 통해 유저 테스트 함수(`user_test_main`)를 실행하도록 작성하였습니다.
