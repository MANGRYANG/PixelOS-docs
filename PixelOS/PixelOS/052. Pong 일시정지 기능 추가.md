# 일일 목표

- [x] `SPACE KEY`를 통해 일시정지할 수 있는 기능 추가 ✅ 2026-05-02

***
# 변경 사항

## 코드

**app/pong.h**

```C

// 게임 상태 관리를 위한 구조체
typedef struct PongState
{
	...
	
    // 게임 일시정지 상태를 저장하는 멤버
    int paused;
    // 이전 프레임에 일시정지 키(SPACE)가 눌려 있었는지 확인하기 위한 멤버
    int space_was_down;
} PongState;
```

게임 상태를 관리하는 구조체 `PongState`에 게임 일시정지 상태를 저장하는 멤버를 추가하였습니다.

일시정지 상태의 토글을 위해 이전 프레임에 일시정지 키가 눌려 있었는지 확인하기 위한 멤버도 추가하였습니다.


**app/pong.c**

```C
// pong 게임 초기화를 위한 내부 함수
__attribute__((section(".usertext")))
static void pong_init(PongState* game)
{
    ...
    
    // 게임 진행: 0, 일시정지: 1
    game->paused = 0;
    game->space_was_down = 0;
}
```

`pong_init` 함수에서 **일시정지 상태**와 **이전 프레임의 일시정지 키 눌림 여부**를 **0으로 초기화**하도록 작성하였습니다.

```C
// pong 게임의 입력 관리를 위한 내부 함수
__attribute__((section(".usertext")))
static void pong_handle_input(PongState* game)
{
	int space_down = app_key_down(APP_KEY_SPACE);
	
    // 일시정지 상태 토글
    // SPACE 키를 누르는 순간 한 번만 실행되도록 제어
    if (space_down && !game->space_was_down)
    {
        game->paused = !game->paused;
    }
	
    // space_was_down 갱신
    game->space_was_down = space_down;
	
    // 게임이 일시정지 상태인 경우 다른 키 입력 무시
    if (game->paused)
    {
        return;
    }
	
	...
}
```

`pong_handle_input` 함수에서 `SPACE BAR` 키를 통해 일시정지 상태를 토글하도록 로직을 추가하였습니다.

```C
__attribute__((section(".usertext")))
static void pong_update(PongState* game)
{
    // 게임이 일시정지 상태인 경우 업데이트하지 않음
    if (game->paused)
    {
        return;
    }
    ...
}
```

`pong_update`에서 게임이 일시정지인 상태인 경우에는 게임 상태를 업데이트하지 않도록 조건문을 추가하였습니다.

```C
__attribute__((section(".usertext")))
static void pong_render(PongState* game)
{
    ...
    
    // 게임이 일시정지 상태인 경우 Paused 메시지 출력
    if (game->paused)
    {
        char pause_text[] = "Paused";
		
        int pause_text_w = 6 * 8;
        int pause_text_h = 16;
		
        int pause_text_x = (game->game_w - pause_text_w) / 2;
        int pause_text_y = (game->game_h - pause_text_h) / 2;
		
        // 텍스트 및 텍스트 배경 박스 출력
        app_game_fill_rect(
	        pause_text_x - 1, pause_text_y - 1,
	        pause_text_w + 2, pause_text_h + 2,
	        COLOR_BLACK
	    );
        app_game_draw_text(
	        pause_text_x, pause_text_y,
	        pause_text,
	        COLOR_WHITE
	    );
    }
	
    // 화면 합성
    app_present();
}
```

`pong_render` 함수에서 게임이 일시정지 상태인 경우 `Paused` 메시지를 출력하는 로직을 추가하였습니다.

텍스트 배경 박스를 함께 출력하여 텍스트 가시성을 안정적으로 확보하도록 구현하였습니다.
