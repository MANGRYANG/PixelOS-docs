# 변경 사항

## 코드

**Makefile**

```
...

$(BUILD_DIR)/kernel.o: $(KERNEL_DIR)/kernel.c
	@echo "==> Compiling kernel..."
	$(CC) -m32 -ffreestanding -fno-builtin \
	-fno-stack-protector -nostdlib -fno-pic \
	-c $(KERNEL_DIR)/kernel.c \
	-o $(BUILD_DIR)/kernel.o

...
```

`Makefile`의 C 소스 컴파일 옵션에 `-fno-pic`를 추가하였습니다.

해당 패치의 근거는 다음 링크를 참조.
https://github.com/MANGRYANG/PixelOS/issues/18

**kernel.c**

```C
...

// 유저 모드에서 실행할 임시 함수 (usertext 섹션)
__attribute__((section(".usertext"), noreturn))
void user_test_main(void)
{
    for (;;) { }
}

...
```

유저 모드 테스트용 임시 함수에서 `hlt` 명령어를 제거하였습니다.

## **리팩토링**

 -  `Makefile` 가독성을 위해 매크로를 통한 리팩토링을 진행하였습니다.
