# 변경 사항

## 코드

**graphics.h**

```C
...

// 백 버퍼가 아닌 VGA에 직접 그리는 함수
void gfx_putpixel_front(int x, int y, uint8_t color);
void gfx_fill_rect_front(int x, int y, int w, int h, uint8_t color);

// 커서를 포함한 프레임 출력 함수
void gfx_present_with_cursor(int cx, int cy, uint8_t cursor_color);
```

기존 프레임 출력 함수인 `gfx_present()`를 **백 버퍼와 커서를 합성하여 프레임을 출력하는 함수**인 `gfx_present_with_cursor()`로 대체하였습니다.

VGA에 직접 렌더링이 필요한 경우를 대비하여, 픽셀을 그리는 `gfx_putpixel_front` 함수와 사각형을 그리는 `gfx_fill_rect_front` 함수를 추가하였습니다.


**graphics.c**

```C
...

void gfx_present_with_cursor(int cx, int cy, uint8_t cursor_color)
{
    const int cw = 8, ch = 8;
	
    for (int y = 0; y < HEIGHT; ++y)
    {
        int row = y * WIDTH;
        for (int x = 0; x < WIDTH; ++x)
        {
            uint8_t c = backbuffer[row + x];
			
            // 커서 영역이면 커서색으로 오버레이
            if (x >= cx && x < cx + cw && y >= cy && y < cy + ch)
                c = cursor_color;
				
            vga[row + x] = c;
        }
    }
}

...

// -- 백 버퍼가 아닌 VGA에 직접 그리는 함수 --
void gfx_putpixel_front(int x, int y, uint8_t color)
{
    if (x < 0 || x >= WIDTH || y < 0 || y >= HEIGHT) return;
    vga[y * WIDTH + x] = color;
}

void gfx_fill_rect_front(int x, int y, int w, int h, uint8_t color)
{
    for (int row = 0; row < h; ++row)
    {
        int current_y = y + row;
        if (current_y < 0 || current_y >= HEIGHT) continue;
		
        for (int column = 0; column < w; ++column)
        {
            int current_x = x + column;
            if (current_x < 0 || current_x >= WIDTH) continue;
            vga[current_y * WIDTH + current_x] = color;
        }
    }
}
```

`gfx_present_with_cursor` 함수는 **백 버퍼 출력 과정에서 커서 영역에 해당하는 범위를 커서의 색으로 오버레이하는 방식**으로 백 버퍼와 커서를 합성하여 프레임을 출력하도록 구현하였습니다.


**cursor.h**

```C
...

void cursor_set_pos(int x, int y);
void cursor_draw(void);

int cursor_get_x(void);
int cursor_get_y(void);
```

**커서의 좌표를 설정**하는 `cursor_set_pos` 함수, **커서를 직접 VGA 프레임버퍼에 그리는 함수**인 `cursor_draw` 함수를 추가하였습니다.

**커서의 x, y 좌표를 반환하는 getter 함수도 추가**하였습니다.


**cursor.c**

```C
...

static int cx = 0, cy = 0;

void cursor_init(int x, int y)
{
    cursor_set_pos(x, y);
}

void cursor_set_pos(int x, int y)
{
    cx = x;
    cy = y;
}

void cursor_draw()
{
    gfx_fill_rect_front(cx, cy, CURSOR_W, CURSOR_H, COLOR_BLACK);
}

int cursor_get_x(void) { return cx; }
int cursor_get_y(void) { return cy; }
```

**기존의 커서 아래 픽셀을 저장하는 로직을 제거**하였으며, 커서 모듈에서는 커서의 좌표 관리만을 담당하도록 분리하였습니다. (`cursor_draw` 함수는 **VGA 프레임버퍼에 직접 커서를 렌더링해야 하는 상황을 대비한 디버깅용 코드**)


**kernel.c**

```C
...

void kernel_main(void)
{   
    ...
	
    int mx = get_mouse_x();
    int my = get_mouse_y();
    cursor_init(mx, my);
    gfx_present_with_cursor(mx, my, COLOR_LIGHT_RED);

    while (1) {
        int new_mx = get_mouse_x();
        int new_my = get_mouse_y();
		
        if (new_mx != mx || new_my != my) {
            cursor_set_pos(new_mx, new_my);
            mx = new_mx;
            my = new_my;
        }
        
        gfx_present_with_cursor(mx, my, COLOR_LIGHT_RED);
    }
}
```

기존 `gfx_present` 함수를 사용한 라인을 `gfx_present_with_cursor` 함수를 사용하도록 수정하였으며, 기존 `cursor_move` 함수도 `cursor_set_pos` 함수로 대체하였습니다.
