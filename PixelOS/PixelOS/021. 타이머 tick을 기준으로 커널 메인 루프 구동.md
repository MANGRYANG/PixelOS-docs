***
# 변경 사항

## 코드

**interrupts.h**

```C
...

extern volatile uint32_t g_timer_ticks;
```

`interrupts.c` 파일에 정의된 `g_timer_ticks`에 다른 C 코드들이 접근할 수 있도록 `extern` 선언을 통해 **전역 심볼로 노출**하였습니다.


**kernel.c**

```C
...

void kernel_main(void)
{   
    ...
	
    while (1) {
        // tick이 바뀔 때까지 대기
        while (g_timer_ticks == last_tick)
        {
            asm volatile("hlt");
        }
		
        ...
    }
}
```

타이머 인터럽트가 발생하여 tick이 바뀔 때마다 CPU를 깨워, **메인 루프가 한 틱 단위로 진행되도록 동기화하는 구조로 변경**하였습니다.
