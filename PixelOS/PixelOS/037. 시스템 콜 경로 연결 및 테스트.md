# 변경 사항

## 코드

**idt.asm**

```ASM
...

global syscall_isr

...

extern syscall_handler

...


;------------------------------------------------------------------
; 시스템 콜 인터럽트 서비스 루틴
;------------------------------------------------------------------
syscall_isr:
    pusha
	
    mov eax, [esp + 28]         ; 시스템 콜 번호 (syscall_num)
    mov ecx, [esp + 24]         ; 시스템 콜 인자 (arg0)
    mov edx, [esp + 20]         ; 시스템 콜 인자 (arg1)
    mov ebx, [esp + 16]         ; 시스템 콜 인자 (arg2)
	
    push ebx                    ; 4번째 인자: arg2
    push edx                    ; 3번째 인자: arg1
    push ecx                    ; 2번째 인자: arg0
    push eax                    ; 1번째 인자: syscall_num
	
    call syscall_handler        ; syscall_handler 호출
	
    add esp, 16                 ; 인자 정리 (스택에서 16바이트 제거)
	
    mov [esp + 28], eax         ; 핸들러의 리턴값을 EAX 레지스터에 저장
	
    popa
	
    iretd
```

**시스템 콜 발생 시 실행되는 ISR을 추가**하였습니다.

시스템 콜 호출 시 **`EAX`에 시스템 콜 넘버(`syscall_num`), `ECX, EDX, EBX`에 시스템 콜 파라미터를 순서대로 전달하도록 ABI를 규정**하였습니다.

**핸들러의 리턴 값**은 **`EAX` 레지스터에 전달**되도록 구현하였습니다.


**interrupts.h**

```C
...

// 시스템 콜 핸들러
uint32_t syscall_handler(uint32_t syscall_no, uint32_t arg0, uint32_t arg1, uint32_t arg2);

...
```

**interrupts.c**

```C
...

extern void syscall_isr(void);

...

// 시스템 콜 넘버 매핑용 Enum
enum
{
    SYS_DEBUG_PUTS = 1,
};

// asm 코드에서 호출되는 시스템 콜 핸들러 정의
uint32_t syscall_handler(uint32_t syscall_no, uint32_t arg0, uint32_t arg1, uint32_t arg2)
{
    (void)arg1;
    (void)arg2;
	
    switch(syscall_no)
    {
        case SYS_DEBUG_PUTS:
        {
            put_string(8, 8, (const char*)arg0, COLOR_LIGHT_GREEN);
            gfx_present();
            return 0;
        }
		
        default:
            return -1;
    }
}

// 인터럽트 초기화 함수
void interrupts_init(void)
{
	...
	
    // 시스템 콜 인터럽트 설정 -> IDT 인덱스 0x80에 매핑됨
    // 시스템 콜 발생 시 syscall_isr 호출
    // 셀렉터(GDT 코드 세그먼트) 0x08
    // type_attributes 0xEE (DPL 3, 32비트 인터럽트 게이트)
    idt_set_gate(0x80, (uint32_t)syscall_isr, 0x08, 0xEE);
	
    ...
}

```

**시스템 콜 핸들러(`syscall_handler`)를 정의**하였습니다.
시스템 콜 테스트를 위한 **문자열 출력 시스템 콜**을 `시스템 콜 넘버 1`으로 설정하였습니다.

**시스템 콜 발생 시 호출되는 ISR(`syscall_isr`)을 IDT에 등록**하였습니다.
시스템 콜 인터럽트(`0x80`)의 권한을 **DPL 3**으로 설정하여, 사용자 어플리케이션이 커널 기능을 요청할 수 있도록 허용하였습니다


**kernel.c**

```C
...

// 유저 모드에서 실행할 임시 함수 (usertext 섹션)
__attribute__((section(".usertext"), noreturn))
void user_test_main(void)
{
    char test_msg[] = "Hello from ring3";
	
    __asm__ volatile (
        "int $0x80"
        :
        : "a"(1), "c"((uint32_t)test_msg), "d"(0), "b"(0)
        : "memory"
    );

    for (;;) { }
}

...
```

유저 모드에서 실행할 임시 함수에서 `SYS_DEBUG_PUTS` 시스템 콜을 호출하여 문자열을 출력하도록 하였습니다.

`syscall_isr`에서 설정한 `ABI`에 맞게 인자를 전달하였습니다.
