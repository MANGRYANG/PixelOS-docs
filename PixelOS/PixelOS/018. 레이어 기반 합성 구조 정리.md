**레이어 관리를 위한 인터페이스를 선언한 헤더 파일**입니다.

**layer_manager.c**

```C
#include "layer_manager.h"

// 최대 레이어 개수 설정
#define MAX_LAYERS 16

// 레이어를 저장할 배열 선언
static Layer* layers[MAX_LAYERS];
static int layer_count = 0;

// 레이어 초기화 함수
void layer_init(void)
{
    layer_count = 0;
}

// 레이어 추가 함수
void layer_add(Layer* layer)
{
    // 최대 레이어 개수를 넘는 레이어는 생성 불가
    if (layer_count >= MAX_LAYERS) return;
    layers[layer_count++] = layer;
}

// 레이어 삭제 함수
void layer_remove(Layer* layer)
{
    for (int i = 0; i < layer_count; ++i)
    {
        if (layers[i] == layer)
        {
            for (int j = i; j < layer_count - 1; ++j)
            {
                layers[j] = layers[j + 1];
            }
            --layer_count;
            
            return;
        }
    }
}

// 레이어 정렬 함수 (Insert Sort)
static void sort_layers(void)
{
    for (int i = 1; i < layer_count; ++i)
    {
        Layer* key = layers[i];
        int j = i - 1;
        while (j >= 0 && layers[j]->z > key->z)
        {
            layers[j + 1] = layers[j];
            j--;
        }
        layers[j + 1] = key;
    }
}

// 레이어 매니저가 관리 중인 모든 레이어를 반환하는 함수
// 인자로 받은 주소에 현재 관리 중인 레이어 수를 저장
Layer** layer_get_all(int* count)
{
    sort_layers();
    *count = layer_count;
    return layers;
}
```

**배열을 통해 레이어 요소들을 관리**하고, `Insert Sort` 방식을 사용하여 **레이어의 `z-index` 오름차순**으로 정렬할 수 있도록 함수를 정의하였습니다.

현재 **레이어 매니저가 관리 중인 레이어 배열의 주소를 반환**하는 함수 `layer_get_all`를 정의하였으며, 인자로 전달받은 `count` 포인터를 통해 **현재 레이어 개수를 호출자에게 전달**합니다.

```C
void wm_composite(void)
{
    // desktop 배경
    gfx_clear(COLOR_LIGHT_GRAY);
	
    int count;
    // 레이어 매니저에서 관리 중인 모든 레이어를 z-index 순서로 읽기
    Layer** layers = layer_get_all(&count);
	
    for (int i = 0; i < count; ++i)
	
    {
        Layer* layer = layers[i];
        // 레이어가 비가시 설정인 경우 다음 레이어로 이동
        if (!layer->visible) continue;
		
        for (int y = 0; y < layer->h; ++y)
        {
            int sy = layer->y + y;
            // y축 범위가 화면 범위를 넘어서는 경우 무시
            if (sy < 0 || sy >= HEIGHT) continue;
			
            for (int x = 0; x < layer->w; ++x)
            {
                int sx = layer->x + x;
                // x축 범위가 화면 범위를 넘어서는 경우 무시
                if (sx < 0 || sx >= WIDTH) continue;
				
                // 레이어의 (x, y) 픽셀을 화면 백 버퍼에 합성
                uint8_t c = layer->buffer[y * layer->w + x];
                gfx_putpixel(sx, sy, c);
            }
        }
    }
}
```

새 윈도우 생성 함수(`wm_create_window`)에서 윈도우 생성 후, `Window` 구조체에 포함된 `layer` 멤버의 정보를 설정하도록 수정하였습니다. 이후 `layer_add`를 호출하여 **해당 윈도우 레이어가 레이어 매니저의 관리 대상에 포함**되도록 설정하였습니다.

`wm_composite`에서 기존의 윈도우 단위 렌더링 대신, 레이어 매니저가 관리 중인 레이어 목록을 `layer_get_all(&count)`로 받아 **z-index 순서대로 순회하며 화면에 합성**하도록 수정하였습니다.
