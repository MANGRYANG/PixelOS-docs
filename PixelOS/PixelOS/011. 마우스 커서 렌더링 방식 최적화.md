# 변경 사항

## 코드

**cursor.h**

```C
#pragma once
#include <stdint.h>

void cursor_init(int x, int y);
void cursor_move(int x, int y);
void cursor_refresh(void);
```

**cursor.c**

```C
#include "cursor.h"
#include "../graphics/graphics.h"
#include "../graphics/color.h"

#define CURSOR_W 8
#define CURSOR_H 8

static int cx, cy;
static uint8_t under[CURSOR_W * CURSOR_H];
static int under_valid = 0;

static inline int idx(int x, int y)
{
    return y * CURSOR_W + x;
}

static void save_under(int x, int y)
{
    for (int j = 0; j < CURSOR_H; ++j)
        for (int i = 0; i < CURSOR_W; ++i)
            under[idx(i, j)] = gfx_getpixel(x + i, y + j);
			
    under_valid = 1;
}

static void restore_under(int x, int y)
{
    if (!under_valid) return;
	
    for (int j = 0; j < CURSOR_H; ++j)
        for (int i = 0; i < CURSOR_W; ++i)
            gfx_putpixel(x + i, y + j, under[idx(i, j)]);
}

static void draw_cursor(int x, int y)
{
    gfx_fill_rect(x, y, CURSOR_W, CURSOR_H, COLOR_BLACK);
}

void cursor_init(int x, int y)
{
    cx = x;
    cy = y;
    save_under(cx, cy);
    draw_cursor(cx, cy);
}

void cursor_move(int x, int y)
{
    if (x == cx && y == cy) return;
	
    restore_under(cx, cy);
    cx = x;
    cy = y;
    save_under(cx, cy);
    draw_cursor(cx, cy);
}

void cursor_refresh(void)
{
    restore_under(cx, cy);
    save_under(cx, cy);
    draw_cursor(cx, cy);
}
```

**커서의 이동과 관련된 인터페이스를 구현**한 C 코드입니다.

- `cursor_init` 함수는 **커서를 초기 위치에 표시하는 함수**입니다.

	커서가 그려질 8x8 픽셀 영역의 기존 화면 픽셀을 `under[]` 버퍼에 저장한 뒤 설정된 위치에 커서를 그리도록 구현하였습니다.

- `cursor_move` 함수는 **커서를 새로운 위치로 이동시키는 함수**입니다.

	**기존 커서 위치의 화면을 원상복구하고 내부 커서 좌표를 갱신**한 뒤, **새 커서가 그려질 8x8 픽셀 영역의 기존 화면 픽셀을 버퍼에 저장하고 커서를 렌더링**합니다.

- `cursor_refresh` 함수는 **커서 위치를 변경하지 않고 화면 상태를 다시 동기화하는 함수**입니다.

	커서 아래의 픽셀이 다른 렌더링 작업으로 인해 변경될 수 있으므로, **현재 커서 위치의 화면을 한 번 원상복구한 뒤 해당 영역을 다시 `under[]` 버퍼에 저장하고 커서를 다시 렌더링**하도록 구현하였습니다.

```C
// 특정 좌표에 위치한 픽셀의 색을 읽어오는 함수
uint8_t gfx_getpixel(int x, int y)
{
    // 범위를 벗어난 픽셀인 경우 무시
    if (x < 0 || x >= WIDTH || y < 0 || y >= HEIGHT)
    {
        return 0;
    }
    // 인자로 주어진 좌표의 픽셀의 색 반환
    return vga[y * WIDTH + x];
}

...
```

그래픽스 엔진에 **특정 좌표에 위치한 픽셀의 색을 읽어오는 함수**를 추가하였습니다.


**kernel.c**

```C
...

void kernel_main(void)
{   
    ...
	
    int mx = get_mouse_x();
    int my = get_mouse_y();
    cursor_init(mx, my);
	
    while (1) {
        int new_mx = get_mouse_x();
        int new_my = get_mouse_y();
		
        if (new_mx != mx || new_my != my) {
            cursor_move(new_mx, new_my);
            mx = new_mx;
            my = new_my;
        }
    }
}
```

기존에는 마우스 좌표 변경 시마다 `wm_draw_all()` 함수를 통해 **화면 전체를 다시 렌더링하는 방식을 사용**하여 비효율적이었습니다.

이에 **초기 커서를 표시한 뒤 좌표가 변할 때만 `cursor_move()` 함수를 통해 커서 부분만 갱신하도록 수정**하여 효율성을 향샹하였습니다.
